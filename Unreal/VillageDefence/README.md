# Village Maker (Village Defence)

> **언리얼 엔진 5 기반의 전략 타워 디펜스 & 서바이벌 액션 게임**

## 1. 프로젝트 소개
- **장르:** 3D 디펜스 + 서바이벌 액션
- **기간:** 2025.10 ~ 2025.12
- **인원:** 2명 (Core System & AI 담당)
- **설명:** 플레이어가 직접 전장에 개입하여 적을 막아내는 '능동적 디펜스' 게임입니다.


## 2. 핵심 구현 기능 (블루프린트)
*(프로젝트 용량 문제로 핵심 로직 스크린샷을 첨부합니다)*

### A. 데이터 기반 타워 시스템
- 타워의 모든 스탯을 데이터 테이블로 관리하여 확장성 확보.


### B. 반응형 적 AI (어그로 스위칭)
- 플레이어 접근 시 타겟을 실시간 변경하고 넥서스로 복귀하는 AI.


### C. 웨이브 & 게임 루프
- 준비/전투 페이즈 자동 순환 및 승리/패배 조건 체크.
